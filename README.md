# 🤖 AI Chat Web - Веб-интерфейс в стиле ChatGPT

Полноценный веб-интерфейс для AI-бота на базе Groq API (LLaMA 3.3 70B)

---

## 📁 Структура проекта

```
ai_chat_web/
├── 📄 server.py              # Flask сервер с API
├── 📄 start.bat              # Быстрый запуск (Windows)
├── 📄 requirements.txt       # Зависимости Python
├── 📄 README.md              # Документация
├── 📁 templates/
│   └── index.html           # HTML интерфейс
└── 📁 static/
    ├── css/
    │   └── style.css        # Стили в стиле ChatGPT
    └── js/
        └── script.js        # JavaScript логика
```

---

## 🚀 Быстрый старт

### Вариант 1: Автоматический запуск (Windows)
```bash
# Просто двойной клик по файлу:
start.bat
```

### Вариант 2: Ручной запуск
```bash
cd ai_chat_web
pip install -r requirements.txt
python server.py
```

**Открой в браузере**: http://localhost:6000

## 📝 Использование

1. **Запусти сервер** через `start.bat` или `python server.py`
2. **Открой браузер** на http://localhost:6000
3. **Выбери режим** в боковой панели слева
4. **Напиши сообщение** в поле ввода
5. **Получи ответ** от AI

### ⌨️ Горячие клавиши
- `Enter` - отправить сообщение
- `Shift + Enter` - новая строка в сообщении
- Кнопка `🗑️` - очистить историю чата
- Кнопка `➕ Новый чат` - начать новый диалог

## 🐛 Решение проблем

### Порт 6000 уже занят
Измени порт в `server.py` (строка 190):
```python
app.run(host='0.0.0.0', port=7000, debug=True)
```

### Ошибка импорта Flask
```bash
pip install Flask openai
```

### API лимит достигнут
Сервер автоматически переключится на следующий ключ из списка (всего 3 ключа).
